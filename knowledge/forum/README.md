# UiPath Forum Knowledge

Dieses Verzeichnis enthält kuratierte Erkenntnisse aus dem UiPath Community Forum (Discourse).

## Struktur

- `latest.json` / `latest.md`: exportierte/zusammengefasste neueste Themen
- `top-<period>.json` / `top-<period>.md`: Top-Themen nach Zeitraum (`daily`, `weekly`, `monthly`, `quarterly`, `yearly`, `all`)
- `search-<query>.json` / `search-<query>.md`: Suchergebnisse und Auswertung
- `topics/<id>.json` / `topics/<id>.md`: Detaildaten einzelner Themen

## Pflegehinweise

- Nutze den MCP-Server `uipath-forum` für Abrufe; speichere relevante Ausgaben hier.
- Bevorzugt strukturierte JSON plus kurze MD-Zusammenfassungen.
- Verlinke in Use Cases auf konkrete Themen/Posts.
