# Use Cases

Dieses Verzeichnis enthält Business-Use-Cases, die als Grundlage für die technische Dokumentation dienen.

## Workflow

1. **Use Case erstellen** → Du legst einen neuen Use Case hier ab
2. **Beauftragung im Chat** → "Dokumentiere bitte UC-001 technisch"
3. **Recherche** → Ich nutze alle MCP-Server für Best Practices
4. **Technische Doku** → Generierung in `knowledge/custom/`

## Namenskonvention

```
uc-001-titel.md           # Use Case Beschreibung
uc-002-titel.md
...
```

## Template

Nutze `usecase-template.md` als Vorlage für neue Use Cases.
